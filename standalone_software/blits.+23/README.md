# Test program for various bitblits
This project tests a set bit blit (bit block transfer) routines. 
## build instructions
To build ```make -ftests.mak```
To cleanup ```make -ftests.mak clean```
To execute the tests ```make -ftests.mak run```
To run clang-tidy ```make -ftests.mak check```
There is no demo program, only tests.
