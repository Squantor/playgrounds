# Test program for various bitblits
This project tests a set bit blit (bit block transfer) routines and their associated helper functions. The 1D routines operate on arrays/spans of ```uint32_t```.
# build instructions
* To build ```make ```
* To cleanup ```make clean```
* To execute the tests ```make run```
* To run clang-tidy ```make check```
There is no demo program, only tests.
